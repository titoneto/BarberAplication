Projeto criado com a finalizade de aprender a desenvolver aplicativos com REACT NATIVE e a utilizar Git

obs: *Primeira vez escrevendo um arquivo README para algum projeto*

O aplicativo já se encontra com desenvolvimento um tanto avançado com as seguintes telas resumidamente:
	
	PreLoad: tela inicial utilizada para verificar se já existe um token disponível (ASYNC storage) 

	SignIn: Tela de login, redirecionada a partir do preLoad caso n seja encontrando nenhum token

	SignUp : Tela de Cadastro

	Home: Tela Principal onde será visualizado os barbeiros 

	Barber: Tela com os informações e serviços do barbeiro, redirecionado a partir da tela HOME 

As demais telas (Search, Appointments, Favorites e Profile) ainda n foram desenvolvidas
